# Evidence

Store captured proof here while the repository is private.

## Folder Convention

```text
screenshots/
logs/
cloudwatch/
step-functions/
sample-responses/
```

Use the lab number as a prefix for every artefact.

Examples:

- `screenshots/02-step-functions-fallback-success.png`
- `logs/04-rag-stale-retrieval.json`
- `cloudwatch/05-tool-circuit-open-metric.png`
